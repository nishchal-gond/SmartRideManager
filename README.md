# 🏍️ SmartRide Manager

![SmartRide Manager Banner](https://github.com/Rio2802/SmartRideManager/blob/main/converted-assasins_creed_force_by_imbattuto8_dj9pgsl.png)

**SmartRide Manager** is a cross-platform React Native + Expo mobile app that helps bike owners track and manage everything about their rides — from maintenance and services to fuel expenses, mileage, and ride records — all in one place.

> 🚧 **Note:** This project is currently **under active development**. New features, UI improvements, and Firebase integrations are being added regularly. Stay tuned for updates!

---

## ✨ Features

- 🧾 **Maintenance Tracking:** Record service history, dates, and costs for every bike.  
- ⛽ **Fuel & Mileage:** Track fuel refills, calculate mileage, and monitor efficiency trends.  
- 💸 **Expense Management:** Log all bike-related expenditures and visualize total costs.  
- 🗓️ **Service Reminders:** Get reminders for upcoming services and oil changes.  
- 🧭 **Ride Records:** Record, view, and analyze rides with detailed statistics.  
- ☁️ **Firebase Integration:** Secure authentication, real-time sync, and Firestore database.  
- 📱 **Offline Support (Coming Soon):** Manage records even without internet access.  

---

## 🧠 Tech Stack

- **Frontend:** React Native (Expo)  
- **Backend:** Firebase (Firestore, Authentication, Cloud Storage)  
- **State Management:** React Context API  
- **Navigation:** React Navigation  
- **UI Framework:** Tailwind CSS (NativeWind)  
- **Testing:** Jest  
- **Deployment:** EAS (Expo Application Services)

## ⚙️ Installation & Setup

1. **Clone this repository:**
   ```bash
   git clone https://github.com/Rio2802/SmartRideManager.git
   cd smartride-manager

   Install dependencies:

npm install


Add your Firebase config:

Create a new Firebase project.

Enable Firestore, Authentication, and Storage.

Copy your Firebase config into:

src/services/firebaseConfig.js


Start the app:

npx expo start


(Optional) Copy .env.example → .env and fill in your Firebase environment variables.
