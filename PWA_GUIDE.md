## üîß Troubleshooting

### Install button doesn't appear

**On Chrome/Edge:**
- Make sure you're using HTTPS (secure connection)
- Try clearing your browser cache
- Check if the app is already installed

**On iOS Safari:**
- The "Add to Home Screen" option is always available in the Share menu
- Make sure you're using Safari (not Chrome on iOS)

### App doesn't work offline

1. Open the app while online first
2. Navigate through a few pages to cache them
3. Check your browser settings allow offline storage
4. Try clearing cache and reinstalling

### Notificationskrajn't working

**On Android:**
- Grant notification permissions when prompted
- Check Chrome notifications are enabled in system settings
- Ensure the app is not in battery optimization mode

**On iOS:**
- Unfortunately, PWA push notifications are very limited on iOS
- This is an Apple restriction, not a limitation of our app
- Consider using the native iOS app for notifications

### Can't find the installed app

**On Android:**
- Check your app drawer
- Look in the "Installed apps" or "PWA" section
- Search for "SmartRide" in your phone's app search

**On Desktop:**
- Check your Start Menu (Windows) or Applications folder (Mac)
- Search for "SmartRide" in your system search
